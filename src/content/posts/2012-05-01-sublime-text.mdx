<p>I’ve created a new category on this blog: <em>My Choices</em>. Here I’ll talk about software, tech.. anything I use and I think It’s worth a try. I don’t want to give a full description or review but just few tips, pros and cons.</p>

<p>Let’s start with a text editor/IDE.</p>

<h2><a href="http://www.sublimetext.com/2" title="Sublime Text 2">Sublime Text 2</a></h2>

<p>I’m always looking for a good text editor that can help me working on simple text files but also on small projects (typically javascript or python). I loved <em>vim</em> (ok, vim is a religion, non an editor ;-) and I continue using it but when I want something of user-friendly it’s not really a good choice.</p>

<p>I start using <a href="http://www.pspad.com/">PSPad</a>, then I switch to <a href="http://notepad-plus-plus.org/">Notepad++</a>, then to <a href="http://www.ultraedit.com/">UltraEdit</a>. Each of these has many features and can be considered a great editor but I think this choice (the editor is one of my most used software) is also a matter of taste.</p>

<p>What I want is:</p>

<ul>
<li>a light but good looking user interface;</li>
<li>cross platform (I switch frequently from windows to linux and maybe in future Mac OS);</li>
<li>programmability (not too complicated, just scripting of executing bat/sh);</li>
<li>column selection;</li>
<li>colour highlighting and formatting for source code;</li>
<li>single file/projects management; After trying for some time Sublime Text 2, I can say it satisfy all these needs. It is one of the fastest editors I’ve tried out, with a light and interference free UI, it’s cross-platform and has a great community of developers that creates plug-ins and themes for any needs.</li>
</ul>

<p>It’s definitely my choice! :-D</p>
